# mi-project
## 项目简介
**简单模仿小米商城**
1. 首页渲染分类信息、商品信息和购物车信息
2. 实现了用户登录，注册功能
3. 实现了详情页的渲染，加入购物车功能
4. 根据用户在购物车页面渲染对应用户添加的商品信息
5. 简单实现了后台管理页面的展示功能
## 项目起手
1. 需要安装npm工具，进入项目根目录，安装项目引用的工具库

    ```shell
    npm -i
    ```
2. 执行项目根目录的 `mi.sql` 数据库文件
3. 在支持php的服务器环境访问项目网站
4. 前台首页：/dist/assets/pages/index.html
5. 后台首页：/dist/admin/pages/index.html
